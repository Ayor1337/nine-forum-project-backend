# 项目概述

## 项目名称
nine-forum-project (九号论坛项目)

## 项目类型
企业级多模块 Spring Boot 论坛应用系统

## 项目目的
这是一个功能完整的论坛系统,支持:
- 用户注册、认证和邮箱验证
- 论坛帖子、回复和主题管理
- 实时聊天功能 (基于 WebSocket)
- 基于角色的权限控制 (管理员、普通用户)
- 文件上传和管理 (通过 MinIO)
- 邮件通知 (通过 RabbitMQ 队列)
- Redis 缓存优化性能
- Elasticsearch 搜索功能

## 技术特点
- 采用多模块 Maven 架构,分离关注点
- 使用 JWT 进行无状态认证
- 支持实时消息推送
- 异步任务处理
- 分布式文件存储

## 部署信息
- 主应用端口: 9966 (web-app)
- 管理后台端口: 9977 (web-admin)
- Java 版本: 17
- Spring Boot 版本: 3.5.5
